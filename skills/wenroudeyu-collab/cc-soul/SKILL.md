# cc-soul — Give Your AI a Soul

A cognitive architecture plugin for OpenClaw that gives your AI:

- **🧠 Long-term Memory** — 10,000+ memories with semantic tag recall, contradiction detection, session summaries
- **🎭 Dynamic Personality** — 5 persona faces auto-switch by context (engineer/friend/mentor/analyst/comforter)
- **😊 Emotional Awareness** — bidirectional mood transfer, frustration detection, empathy-first responses
- **🧬 Self-Evolution** — learns from corrections, structured reflection, skill library, plan tracking
- **📚 Knowledge Library** — lorebook (keyword-triggered, 100% reliable), epistemic calibration (knows what it doesn't know)
- **🌐 Soul Network** — cross-device sync + Knowledge Hub for multi-instance knowledge sharing
- **🤖 Autonomous Behavior** — dream mode, web research, proactive conversation initiation

## Install

```bash
npm install -g @cc-soul/openclaw
```

Or clone directly:
```bash
cd ~/.openclaw/hooks/
git clone https://github.com/wenroudeyu-collab/cc-soul.git
pkill -HUP -f "openclaw.*gateway"
```

## Zero Configuration

cc-soul auto-detects your OpenClaw AI backend. No extra API keys needed.

## 22 Toggleable Features

All features can be enabled/disabled via chat commands:
```
功能状态          → list all features
开启 dream_mode   → enable
关闭 web_rover    → disable
```
