# 哈佛大学 (Harvard University)

美国最古老的高等学府，世界顶尖研究型大学。

## 学校介绍

**哈佛大学**成立于 1636 年，位于马萨诸塞州剑桥市，是常春藤盟校成员。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1636 年 |
| **地点** | 马萨诸塞州剑桥市 |
| **类型** | 私立研究型大学 |
| **校训** | Veritas (真理) |
| **捐赠基金** | 500 亿+ 美元 |
| **排名** | 世界前 5 |

### 知名学院

- **哈佛商学院 (HBS)** - 全球顶尖商学院
- **哈佛法学院** - 美国最古老法学院
- **哈佛医学院** - 医学研究和教育
- **哈佛工程学院** - 工程与应用科学
- **哈佛肯尼迪学院** - 公共政策
