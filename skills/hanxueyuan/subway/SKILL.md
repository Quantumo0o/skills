# 赛百味 (Subway)

全球最大的三明治连锁品牌。

## 品牌介绍

**赛百味**成立于 1965 年，总部位于美国康涅狄格州，是全球最大的快餐连锁品牌之一。

### 基本信息

| 项目 | 详情 |
|------|------|
| **成立时间** | 1965 年 |
| **创始人** | 弗雷德·德卢卡 |
| **总部** | 美国康涅狄格州米尔福德 |
| **门店数量** | 37000+ 家 |
| **覆盖国家** | 100+ |

| 产品 | 介绍 |

### 特色服务

- **自定义搭配**: 面包、肉类、蔬菜、酱料自选
- **新鲜制作**: 现场制作保证新鲜
- **健康选择**: 低卡、高蛋白选项
- **早餐服务**: 早餐三明治、咖啡
- ** catering**: 大型活动订餐
