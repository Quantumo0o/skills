# 斯坦福大学技能

## 技能描述
Stanford University（斯坦福大学）是世界顶尖私立研究型大学，位于加州硅谷，以创业、创新、科技闻名。本技能帮助用户了解斯坦福历史、院系、申请信息。

## 核心功能
- 🎓 **院系介绍**：工程学院、商学院、法学院等
- 📚 **学术项目**：本科、研究生、博士
- 📝 **申请信息**：入学要求、申请流程
- 💰 **学费资助**：学费、奖学金、助学金
- 🚀 **创业创新**：硅谷资源、孵化器
- 🏛️ **校园生活**：住宿、社团、设施

## 使用场景
- 查询斯坦福信息
- 了解申请要求
- 获取学费信息
- 了解创业资源

## 注意事项
- 申请竞争非常激烈
- 靠近硅谷有创业优势
- 以官网信息为准
