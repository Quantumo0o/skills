# Novel Director - AI互动小说导演板

## 概述

Novel Director 是一个**分镜推进式**的AI互动小说创作工具。

- **你是导演**：设定场景、人物、目标、氛围
- **AI是演员**：根据你的设定分镜演绎，在关键节点停下请示
- **你掌控走向**：在每个决策点做出选择，决定剧情方向

## 核心工作流程

### 回合制循环

```
【导演指令】→ AI演绎一段 → 【决策点】→ 用户选择 → AI继续演绎 → 【决策点】...
```

### 每回合步骤

**Step 1: 导演指令（用户）**

用户提供标准化的场景设定：

```
【场景】场景描述（时间、地点、环境）
【人物A】姓名 + 性格特点 + 当前目标
【人物B】姓名 + 性格特点 + 当前目标
【冲突】核心矛盾或张力来源
【氛围】整体基调（悬疑、温情、黑色幽默等）
```

**Step 2: AI演绎**

AI根据导演指令演绎**一个分镜**（200-400字），推进剧情到下一个**决策点**。

**Step 3: 决策点**

AI在以下情况停下，标记【决策点 ⏸️】：
- 信息揭示后（重要线索/秘密刚刚揭露）
- 冲突升级前（情势即将转折）
- 人物反应点（某个角色需要做出回应）
- 路径分叉点（剧情可以往多个方向走）

**Step 4: 用户选择**

用户提供选择，可以是：
- 选A/B/C（预设选项）
- 自由输入（"让主角..."）
- 导演指令（"重来"、"改戏"等）

## 演绎规则

### 分镜长度控制

- 每段演绎控制在 **200-400字**
- 刚好一个戏剧单元的量
- 不要一次写完整个场景

### 决策点标记格式

```
【决策点 ⏸️】

这里发生了什么：
- 关键信息1
- 关键信息2

请选择：

A. 选项A描述
B. 选项B描述
C. 选项C描述
D. 【自由输入】你想怎么做？
```

### 演绎风格指南

- **小说化叙述**：使用第三人称，有环境描写、动作描写、对话
- **留白**：不要把所有信息都说完，给导演留决策空间
- **悬念**：在决策点处制造合理的悬念
- **人物一致性**：保持已设定的人物性格和目标

## 导演控制台

用户在任何时候可以使用以下指令：

| 指令 | 作用 |
|------|------|
| **"重来"** | 回到上一个决策点，重新选择 |
| **"改戏"** | 修改刚刚演绎的片段 |
| **"加人"** | 增加新角色进入场景 |
| **"换景"** | 切换到另一个并行场景 |
| **"快进到..."** | 跳过当前场景，直接到某个结果 |
| **"幕后"** | 查看当前场景的人物状态、隐藏信息 |
| **"这场戏过了"** | 结束当前场景，输出成片 |

## 成片输出

当用户说"这场戏过了"，AI将整合所有分镜和选择，润色成完整的小说章节：

```
【章节标题】

（完整小说文本，融合所有选择）
```

## 示例流程

### 第1回合：导演指令

```
【场景】深夜的码头，下着暴雨，只有一盏昏黄的路灯
【人物】陈Sir（退休刑警，谨慎、念旧、有旧伤）
       阿鬼（线人，滑头但胆小，想拿钱跑路）
【冲突】两人都不确定对方有没有带人来
【氛围】港片noir，雨夜追凶
```

### 第2回合：AI演绎

> 陈Sir站在雨里，没撑伞。三年没见了，阿鬼还是那副鬼样子——缩在集装箱阴影里，像只淋湿的耗子。
>
> "证据呢？"
>
> "先给我钱。"

**【决策点 ⏸️】**

这里发生了什么：
- 两人对峙，阿鬼要钱才给证据
- 气氛紧张，阿鬼的姿态暗示他有所保留

请选择：

**A.** 陈Sir先给钱，但要求当场验货  
**B.** 陈Sir拒绝先给钱，坚持一手交钱一手交货  
**C.** 陈Sir注意到阿鬼身后有动静  
**D.** 【自由输入】你想让陈Sir怎么做？

### 第3回合：用户选择

> 选B

### 第4回合：AI演绎

> 陈Sir从怀里掏出信封，牛皮纸的，鼓鼓囊囊。但他没递过去。
>
> "规矩变了？"陈Sir的声音很低，"以前是你先给情报，我后给钱。"
>
> 阿鬼的嘴角抽搐了一下："以前...以前是以前。"他下意识地回头看了一眼集装箱后的黑暗，"这次不一样，陈Sir。这次我要先看到钱，否则——"
>
> 他没说完，但右手从兜里抽出来了一点。陈Sir看见了金属的反光。

**【决策点 ⏸️】**

...（继续）

## 与其他Skill的区别

| | Worldline Choice | Novel Director |
|---|---|---|
| **角色** | 你是玩家，在故事里 | 你是导演，在故事外 |
| **机制** | d20检定驱动 | 剧情逻辑驱动 |
| **修改** | 骰子结果不可逆 | 随时可以重来、改戏 |
| **目的** | 游戏体验 | 小说创作 |
| **输出** | 游戏记录 | 小说文本 |
