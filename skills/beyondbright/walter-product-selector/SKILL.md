# 亚马逊选品方向分析

## 核心目标

帮助用户找到可执行的选品方向，基于12大选品方法论三层决策逻辑。

## 用户输入

- 品类方向或种子词（可选）
- 目标市场（美国/欧洲/日本）
- 预算范围（可选）
- 经验水平（新手/有经验）

## 12大选品方法论

### 第一层：发现机会

| 方法 | 说明 | 数据来源 |
|------|------|---------|
| 榜单选品 | BS / New Releases / Movers & Shakers / Most Wished For / Gift Ideas | 亚马逊前台 |
| 社交趋势 | TikTok/Reddit/YouTube爆款趋势 | 站外社媒 |
| 新类目节点 | 亚马逊新开的小类目/细分节点 | 亚马逊类目树 |
| 下拉词 | 亚马逊搜索框自动补全关键词 | 亚马逊前台 |
| 竞品店铺 | 监控目标竞品的新品动态 | 竞品店铺 |
| 站外趋势 | Reddit等社区讨论热度 | Reddit |
| 冷门蓝海 | 高门槛品类（FDA/510K认证/医疗器械） | 专业渠道 |

### 第二层：验证可行性

| 方法 | 说明 | 核心指标 |
|------|------|---------|
| 市场容量/广告经济性 | 谷歌趋势+CPC/CPA/广告占比/利润率/退款率 | SPR、ACOS |
| VOC痛点挖掘 | 评论分析→挖掘痛点→发现未满足需求→创新 | 差评/QA |
| 差异化对标 | 分析竞品为何卖得好→对比自身优势→找出差异 | 差异化指数 |

### 第三层：产品定义

| 方法 | 说明 |
|------|------|
| 元素迁移 | 拆解爆款元素标签，像搭乐高一样重新组合 |
| 场景/人群/情绪升维 | 从比价格→比场景/比人群/比情绪价值 |

## 三层决策逻辑

```
┌────────────────────────────────────────────────┐
│              第一层：发现机会                     │
│   榜单选品 / 社交趋势 / 新类目节点 / 下拉词     │
│   → 回答：做什么方向？哪里有机会？               │
└─────────────────────┬──────────────────────────┘
                      ▼
┌────────────────────────────────────────────────┐
│              第二层：验证可行性                   │
│   市场容量 / VOC痛点 / 差异化对标               │
│   → 回答：能不能做？赚不赚钱？有没有差异化空间？ │
└─────────────────────┬──────────────────────────┘
                      ▼
┌────────────────────────────────────────────────┐
│              第三层：产品定义                     │
│   元素迁移 / 场景人群情绪升维                   │
│   → 回答：做成什么样的产品？                    │
└────────────────────────────────────────────────┘
```

## 元素迁移法（三步走）

### 第一步：全维解构 (Deconstruction)

建立标准化标签体系：

| 层级 | 标签维度 |
|------|---------|
| 基础身份层 | 品类、主题/IP、目标用户 |
| 视觉感官层 | 风格、材质、色系、形状 |
| 功能价值层 | 硬功能（功率/噪音/触控等）、软价值（装饰/礼品） |

### 第二步：数据验证 (Validation)

| 组合 | 含义 | 策略 |
|------|------|------|
| 高销量 + 高频元素 | 核心积木 | 必须有 |
| 高销量 + 低频元素 | 蓝海差异点 | 机会！ |
| 低销量 + 高频元素 | 过度同质化 | 避开 |
| 低销量 + 低频元素 | 无效元素 | 忽略 |

### 第三步：重组创新 (Reconstruction)

```
爆款公式：[热门赛道] + [高热度情感内核] + [高价值功能] = 新品
```

## 输出格式

### 选品方向报告

| # | 方向 | 方法来源 | 机会类型 | 风险提示 | 推荐指数 |
|---|------|---------|---------|---------|---------|
| 1 | ... | 榜单选品 | 蓝海 | 低门槛 | ★★★★☆ |

### 每个方向的详细分析

```
选品方向：[具体品类/切入角度]

方法论依据：来自12大方法中的[具体方法]
机会描述：[为什么是好方向]
目标人群：[核心用户画像]
预估市场：[容量/趋势]
风险提示：[需要注意的问题]
下一步行动：[进入市场可行性分析]
```

## 注意事项

- 选品方向只是起点，具体可行性需要进入 walter-market-intelligence 验证
- 不确定时，提供2-3个备选方向供用户选择
- 优先推荐有差异化空间的方向
- 高门槛方向需评估用户自身能力匹配度
