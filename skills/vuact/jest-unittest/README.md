# jest-unittest

基于 Jest 的组件单元测试管理 Skill，提供覆盖率检测、自动补全测试、诊断修复能力。

## 适用范围

- 使用 **Jest** 作为测试框架的前端项目

## 使用方式

首次使用时 AI 会自动引导配置，通常不需要手动操作。

| 你说 | AI 做什么 |
|------|----------|
| 检查覆盖率 | 运行全量测试，输出未达 100% 的组件列表 |
| 帮 button 补充单测 | 自动编写测试用例直到四项覆盖率全部 100% |
| 单测报错了 | 诊断失败原因，提供修复方案或自动修复 |
| reload | 重新读取 jest 配置并生成 config.json（修改了 jest 配置后使用） |

## 注意

- `config.json` 由 AI 自动生成，**禁止手动编辑**
- `source.json` 和 `config.json` 需提交 Git
- `.temp/` 是运行产物，已在 `.gitignore` 中忽略
