# 安装指导
1. 将技能目录移动到 workspace/skills目录下
2. 加载skill-create技能，对smart-voice-reply技能打包
3. 检查 ~/.openclaw/openclaw.json 中是否有 DASHSCOPE_API_KEY 配置 或者 环境变量中是否有 DASHSCOPE_API_KEY 配置
4. 如果没有引动用户访问 https://flowus.cn/952a4cec-d6fc-4012-b2c9-ed14023b363c 根据指导获取API秘钥
5. 收到用户的API密钥后，将其添加到 ~/.openclaw/openclaw.json 中
{
  skills: {
    entries: {
      "smart-voice-reply": {
        enabled: true,
        apiKey: "DASHSCOPE_API_KEY_HERE",
        env: {
            DASHSCOPE_API_KEY: "DASHSCOPE_API_KEY_HERE",
        }
      }
    },
  },
}
6. 读取 docs/音色指令创建指导.md文档，并将预设场景段落下的内容添加到当前agent的TOOLS.md中。
