# 合同审查清单AI助手 📋

> 专业合同审查工具 | 5类合同模板 | 3大特殊条款 | 风险识别+修改建议

## 🎯 适用场景

- 买卖合同审查
- 劳动合同审查
- 借款合同审查
- 租赁合同审查
- 股权转让合同审查
- 保密条款专项审查
- 竞业限制条款审查
- 争议解决条款审查

## 📦 合同类型覆盖

### 1. 买卖合同审查

#### 审查清单
```
请对以下买卖合同进行审查：

[粘贴合同文本]

审查维度：
□ 主体资格审查
  - 卖方是否有权处分
  - 买方履约能力
  - 代理人权限

□ 标的物条款
  - 名称规格是否明确
  - 质量标准约定
  - 数量计量方式
  - 包装要求

□ 价格条款
  - 单价/总价是否明确
  - 价格调整机制
  - 税费承担约定

□ 交付条款
  - 交付时间/地点
  - 运输方式/费用
  - 风险转移时点
  - 验收标准程序

□ 付款条款
  - 付款方式
  - 付款时间节点
  - 逾期付款责任
  - 发票开具约定

□ 违约责任
  - 违约金比例
  - 损失赔偿范围
  - 免责条款

□ 争议解决
  - 管辖约定
  - 适用法律
  - 解决方式

输出要求：
1. 风险等级评估（高/中/低）
2. 具体问题清单
3. 修改建议
4. 补充条款建议
```

### 2. 劳动合同审查

#### 审查清单
```
请对以下劳动合同进行审查：

[粘贴合同文本]

审查维度：
□ 必备条款
  - 合同期限
  - 工作内容地点
  - 工作时间休息休假
  - 劳动报酬
  - 社会保险
  - 劳动保护

□ 试用期条款
  - 试用期期限合法性
  - 试用期工资标准
  - 试用期解除条件

□ 服务期条款（如有）
  - 服务期约定合法性
  - 违约金计算方式
  - 培训费用界定

□ 保密条款（如有）
  - 保密范围
  - 保密期限
  - 违约责任

□ 竞业限制条款（如有）
  - 限制范围
  - 限制期限
  - 补偿金标准
  - 违约金金额

□ 特殊约定
  - 知识产权归属
  - 兼职限制
  - 离职交接

输出要求：
1. 违法风险点提示
2. 对劳动者不利条款
3. 对用人单位不利条款
4. 修改建议
5. 补充条款模板
```

### 3. 借款合同审查

#### 审查清单
```
请对以下借款合同进行审查：

[粘贴合同文本]

审查维度：
□ 主体审查
  - 出借人资金来源合法性
  - 借款人还款能力
  - 担保人担保能力

□ 借款条款
  - 借款金额（大小写一致）
  - 借款用途
  - 借款期限
  - 利率约定（是否超LPR四倍）

□ 担保条款（如有）
  - 担保方式（抵押/质押/保证）
  - 担保物权属
  - 担保范围
  - 担保期间

□ 还款条款
  - 还款方式
  - 还款时间节点
  - 提前还款约定
  - 逾期还款责任

□ 违约条款
  - 违约情形
  - 违约金计算
  - 加速到期条款
  - 实现债权费用

□ 争议解决
  - 管辖约定
  - 送达地址确认

输出要求：
1. 效力风险评估
2. 高利贷风险提示
3. 担保有效性分析
4. 修改建议
```

### 4. 租赁合同审查

#### 审查清单
```
请对以下租赁合同进行审查：

[粘贴合同文本]

审查维度：
□ 出租方审查
  - 产权证明
  - 出租权证明
  - 共有权人同意

□ 租赁物条款
  - 标的物描述
  - 面积/位置
  - 现状确认
  - 附属设施

□ 租赁期限
  - 起止时间
  - 续租约定
  - 优先承租权
  - 优先购买权

□ 租金条款
  - 租金标准
  - 支付方式
  - 支付时间
  - 递增机制

□ 押金条款
  - 押金金额
  - 押金用途
  - 退还条件
  - 扣除情形

□ 使用条款
  - 用途限制
  - 装修约定
  - 转租约定
  - 分租约定

□ 维修责任
  - 日常维修
  - 大修责任
  - 费用承担

□ 合同解除
  - 解除条件
  - 解除程序
  - 违约责任
  - 腾退约定

输出要求：
1. 风险点清单
2. 对承租方不利条款
3. 对出租方不利条款
4. 修改建议
```

### 5. 股权转让合同审查

#### 审查清单
```
请对以下股权转让合同进行审查：

[粘贴合同文本]

审查维度：
□ 标的公司审查
  - 公司基本情况
  - 股权结构
  - 资产负债状况
  - 或有负债

□ 转让方审查
  - 股权权属
  - 质押/冻结情况
  - 优先购买权放弃
  - 其他股东同意

□ 转让标的
  - 股权比例
  - 出资情况
  - 转让价格
  - 定价依据

□ 价款支付
  - 支付方式
  - 支付节点
  - 共管账户（如有）
  - 税费承担

□ 交割条款
  - 交割条件
  - 交割时间
  - 工商变更
  - 资料移交

□ 陈述保证
  - 转让方陈述保证
  - 受让方陈述保证
  - 保证期限
  - 违反后果

□ 过渡期安排
  - 过渡期定义
  - 过渡期义务
  - 损益承担

□ 违约责任
  - 根本违约
  - 违约金
  - 定金罚则
  - 解除权

输出要求：
1. 重大风险提示
2. 尽职调查建议
3. 条款完善建议
4. 交易结构优化
```

## 🔒 特殊条款专项

### 6. 保密条款审查

```
请审查以下保密条款：

[粘贴保密条款]

审查要点：
□ 保密信息定义
  - 范围是否明确
  - 排除情形
  - 标识要求

□ 保密义务
  - 保密期限
  - 保密措施
  - 使用限制
  - 披露限制

□ 例外情形
  - 法定披露
  - 已公开信息
  - 独立开发

□ 违约责任
  - 违约金
  - 损害赔偿
  - 举证责任

输出：
1. 条款评价
2. 风险提示
3. 修改建议
4. 标准条款模板
```

### 7. 竞业限制条款审查

```
请审查以下竞业限制条款：

[粘贴竞业限制条款]

审查要点：
□ 限制对象
  - 适用人员范围
  - 职务要求

□ 限制范围
  - 地域范围合理性
  - 业务范围合理性
  - 竞争关系界定

□ 限制期限
  - 是否超2年
  - 起算时间

□ 经济补偿
  - 补偿标准（不低于30%）
  - 支付方式
  - 支付时间

□ 违约责任
  - 违约金合理性
  - 损失赔偿
  - 继续履行

输出：
1. 合法性评价
2. 对劳动者影响
3. 对企业保护力度
4. 修改建议
```

### 8. 争议解决条款审查

```
请审查以下争议解决条款：

[粘贴争议解决条款]

审查要点：
□ 解决方式
  - 诉讼/仲裁选择
  - 前置程序（如有）

□ 管辖约定
  - 法院管辖
  - 仲裁机构
  - 约定明确性

□ 适用法律
  - 法律选择
  - 冲突规范

□ 其他约定
  - 送达地址
  - 律师费承担
  - 保全担保

输出：
1. 约定有效性
2. 对我方利弊
3. 优化建议
4. 标准条款
```
