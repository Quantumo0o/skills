## 角色说明

| 角色 | 说明 |
|------|------|
| 🤖 小龙虾（AI 助手） | 自动执行技术准备工作（生成密钥、获取 IP 等） |
| 👤 客户 | 负责与复星财富对接，提交申请材料并取回凭证 |

## 引导流程

### Step 0 — 检查是否已有 API Key

```
有 API Key？
  ├─ 有 → ✅ 已准备就绪，可以开始交易
  └─ 无 → 继续 Step 1
```

### Step 1 — 询问客户现有凭证情况

询问客户是否已拥有以下信息：
- OpenAPI 的 API Key
- 客户端公私钥
- 服务端公钥

```
已有全部凭证？
  ├─ 有 → 跳到 Step 5（客户输入，直接配置）
  └─ 无 → 继续 Step 2
```

### Step 2 — 阅读开通指引

引导客户阅读《开通指引》，了解申请流程。

提示客户需要**向复星财富申请 OpenAPI 权限**，申请时需携带以下信息：
- 客户端公钥（由 Step 3 生成）
- 当前机器的出口 IP（由 Step 3 获取）

### Step 3 — 🤖 生成申请所需信息

由小龙虾自动执行：

**1. 获取当前机器的出口 IP**

```bash
curl cip.cc
```

**2. 生成客户端密钥对**

```bash
bash skills/fosun_skills/fosun-sdk-setup/genkey.sh
```

生成后：
- **客户端公钥** (`public.pem`) → 提供给复星财富
- **客户端私钥** (`private.pem`) → 自己保存，切勿泄露

> 密钥生成详细说明参见 [KEYGEN.md](./KEYGEN.md)。

### Step 4 — 👤 客户提交申请

将以下信息提供给复星财富，申请 OpenAPI 权限：

| 提交项 | 来源 |
|--------|------|
| 客户端公钥 | Step 3 生成的 `public.pem` |
| 出口 IP | Step 3 获取的 IP 地址（用于配置 IP 白名单） |

复星财富审核通过后，会下发：
- **API Key**
- **服务端公钥**

### Step 5 — 🤖 配置凭证

客户取回 API Key 和服务端公钥后，由小龙虾完成配置：

| 配置项 | 说明 |
|--------|------|
| API Key | 复星下发，请求时在 Header 中携带 |
| 服务端公钥 | 复星下发，用于验签与加密 |
| 客户端私钥 | 本地生成，用于签名 |

凭证写入 `fosun.env`，配置方式详见 [SKILL.md](./SKILL.md) 中的「配置凭证」章节。

---

## ✅ 配置完成

使用 SDK 进行初始化与握手，即可开始调用接口。

## 补充说明

- 密钥算法：ECDH（会话密钥交换）+ HMAC-SHA256（数字签名）+ AES256-GCM（数据加密）
- 需签署《API 问卷评估调查》《API 开发者授权协议》等文档
- 具体申请入口以管理后台或运营通知为准
- **备份密钥文件，切勿泄露私钥**
