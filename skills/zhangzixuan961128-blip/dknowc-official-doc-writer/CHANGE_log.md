# 更新日志

## [2.0.1] - 2026-04-10

### 安全修复
- `dkag_search.py`：不再读取 `~/.openclaw/openclaw.json` 完整配置文件，改用 `os.environ.get()` 读取环境变量
- 清理残留的冗余子目录

## [2.0.0] - 2026-04-10

### 全新重构
- 对齐本机版核心能力，完整重写 SKILL.md 流程
- 素材管理全流程：搜索→分类→自检→确认→范文召回→撰写→生成Word
- 素材分类标准：新增来源级别（国家级/本省 vs 其他省市），多地域搜索按地域隔离展示
- 素材充分性自检：AI自动评估政策依据/数据支撑/参考案例覆盖度，不足则自主补搜
- policyFiles按需补充：撰写时发现政策引用缺少文件全称/发文字号，自主补搜
- 知识专库链接即时发送
- 搜索词构造规则和自检标准
- 引用示例通用化，移除所有具体单位名称

### Scripts
- `dkag_search.py`：新增 --policy 参数（返回规范性文件清单）；API地址可配置
- `format_document.py`：新增超链接支持、主送机关长度过滤、页边距四边分别配置
- `template_generator.py`：支持会议纪要红头/表尾模板、WPS文本框占位符替换
- 新增 `merge_search_results.py`：批量搜索结果合并

### Reference Standards
- 16个文种标准文件（15种法定公文 + 事务文书模板）
- 常用句式和结构参考
- 格式规范引用 config/format.json

- Word排版（普通格式/红头格式）
